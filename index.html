This site uses Just the Docs, a documentation theme for Jekyll.

This Serverless Literature Guide aims to provide an overview of a once-niche field that has grown rapidly. A quick search on DBLP shows how the number of publications about serverless has grown year-by-year.

Year Publications
2016 9
2017 31
2018 68
2019 137
2020 176
2021 236

Our aim is to include peer-reviewed publications in prominent venues, as well as other work that has attracted substantial attention. A project of this nature is destined to be out-of-date almost the moment it is published, and we therefore encourage the community to contribute additions by submitting pull requests to the GitHub repository.
